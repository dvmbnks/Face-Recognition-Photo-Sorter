# Face Recognition Photo Sorter

## Description
This application allows users to sort photos based on face recognition. Users can capture a reference image or select one from their files, and then scan a folder of images to find matches. Matching images are copied to a specified output folder.

## Requirements
To run this project, you need to install the following Python packages:

1. opencv-python
2. face_recognition
3. Pillow

You can install these packages using pip:

```bash
pip install opencv-python
pip install face_recognition
pip install Pillow
